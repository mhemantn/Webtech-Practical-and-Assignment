3. Lists

HTML supports several list types to organize content.

Unordered List (ul) - Bullet Points

Used when the order of items is **not** important.

  • Apples
  • Bananas
  • Cherries

Ordered List (ol) - Numbered Items

Used when the order of items **is** important (e.g., steps or rankings).

  1. First Step: Prepare the ingredients.
  2. Second Step: Mix the dry and wet components.
  3. Third Step: Bake according to the recipe.

Nested List (A List within a List)

Used to create hierarchical structures.
