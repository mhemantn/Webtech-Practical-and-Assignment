1. The CSS Box Model

Every element on a web page is a rectangular box. The Box Model dictates how this box is structured, determining its size, spacing, and appearance.

Margin

Border

Padding

Content

Starting from the inside out: Content is surrounded by Padding, which is surrounded by a Border, and finally, Margin creates space outside the border.

2. Margins and Padding

This is the content inside the box.

Code Example:

padding: 20px 40px; /* Top/Bottom: 20px, Left/Right: 40px */
margin: 30px 15px; /* Top/Bottom: 30px, Left/Right: 15px */

Key Differences:

  • Padding: Controls the space inside the box, between the content and the border. It takes the element's background color.
  • Margin: Controls the space outside the box, between elements. It is always transparent.

3. Borders and Background Images
